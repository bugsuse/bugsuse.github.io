Python

  • 配色库

    • colormap: 用于颜色类型转换以及构建colormap
    • cmocean: 海洋科学绘图colormap，除python版外，还有对应的其他版本
    • colorcet: 提供了很多可选的colormap
    • palettable: 支持很多颜色模式

  • 气象相关库

    • Metpy：气象数据读取，计算和可视化工具

    • Sharppy：探空和风场分析工具

    • Siphon：远程数据下载工具

    • salem：地球科学数据处理和绘图库，其中提供了WRF模式相关函数，比如WPS绘图，垂直插值等。

    • wrf-python：WRF模式后处理库

  • 地图相关库
